Dependencies:
Python 3.5 or higher (if you have Python3.x installed, x < 5, you need to upgrade Python)
Microsoft Visual Studio (to be installed by Admin)

Python 3.5 or higher: one of the following options as a user (no admin credentials necessary)
if already installed as admin      -> run setup_admin.bat
if already installed as local user -> run setup_local.bat
if no python installed             -> run setup_local.bat

(Remark: We use a dedicated environment for installing the prototype)

Prototype:
run prototype.bat to calculate optimal points and open navigator

Folder structure:
Desktop\FORCE-BDSS-Prototype\force-bdss-plugin-example
where force-bdss-plugin-example is the github repository.
